Privacy Policy – Learn Anything GPT

Effective Date: 12 February 2026

Overview

Learn Anything GPT uses an external backend API to store learning progress so users can resume lessons across sessions.

What Data Is Stored

  • A randomly generated Learner ID (UUID)
  • Lesson attempts
  • Quiz scores
  • Weak topic tags
  • Timestamps of learning activity

What Data Is NOT Collected

  • No names
  • No email addresses
  • No personal identifiers
  • No payment information
  • No marketing tracking

How Data Is Used

Stored data is used solely to:

  • Resume learning progress
  • Adapt lesson difficulty
  • Provide spaced repetition and revision

Data Sharing

No user data is sold, shared, or transferred to third parties.

Data Deletion

Users may request deletion of their learning record by contacting: your-email@example.com

Security

Internal administrative endpoints are protected and not accessible to users. Reasonable technical measures are used to prevent unauthorized access.
